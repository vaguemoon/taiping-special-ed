# 換算趣 — 檔案功能說明

> appId: `convert`  
> 類型: 遊戲類（含成就系統）  
> 最後更新: 2026-04

## 目錄結構

```
convert/
├── index.html          ← App 入口（6 個頁面）
├── convert.css         ← App 專屬樣式
├── 檔案功能說明.md     ← 本檔案
└── js/
    ├── nav.js          ← 頁面切換與 PAGE_STACK
    ├── state.js        ← 全域狀態與 Firestore 寫入
    ├── questions.js    ← 三大題域題目生成器
    ├── game.js         ← 選擇頁渲染與遊戲迴圈
    ├── achievement.js  ← 成就系統、稱號、每日登入
    └── init.js         ← 啟動、自動登入、設定頁（最後載入）
```

## 各 JS 職責

### nav.js
- `PAGE_STACK` 導覽歷史陣列
- `PAGE_CONFIG` 各頁面的標題與返回鍵設定
- `showPage(name, pushHistory)` — 切換頁面，更新 Topbar
- `goBack()` — 返回上一頁
- `backToHub()` — postMessage 通知 Hub 返回

### state.js
- `currentStudent` — 目前登入學生物件
- `currentCategory / currentSubtype / currentDifficulty` — 遊戲設定
- `gamePool / gamePoolIdx / gameQ` — 一輪題目狀態
- `fillInputStr / fillInputArr / activeFillIdx / answerCount` — 輸入狀態
- `totalCorrect / totalRounds / bestStreak / categoryStats` — 歷史統計
- `saveProgress()` → `students/{id}/progress/convert`
- `shuffle(arr)` — Fisher-Yates 洗牌

### questions.js
- `generateQuestionPool(category, subtype, difficulty)` → 洗牌後取前 10 題
- 長度題型：`mm-cm`、`cm-m`、`m-km`、`mixed`
- 時間題型：`to-min`（小時分→分）、`to-hm`（分→小時分，雙答）、`mixed`
- 貨幣題型：`exchange`（換錢反射）、`optimal`（台幣最省硬幣）、`mixed`
- 題目物件格式：`{ prompt, answerCount: 1|2, answer: [number,...], correctText }`

### game.js
- `openCategory(cat)` — 進入選擇頁
- `renderSelectPage()` — 動態渲染選擇頁（題型 + 難度按鈕）
- `startGame()` — 初始化一輪遊戲，呼叫 `generateQuestionPool`
- `loadQuestion()` — 載入下一題或跳到結果頁
- `renderQuestion()` — 顯示題目；單答/雙答切換不同輸入區
- `fillAppend / fillBackspace / onGameSubmit` — 虛擬鍵盤邏輯
- `onGameResult(isCorrect)` — 答題結果處理、音效、計分
- `showResult()` — 計算星評（≥90%→3星）並顯示結果頁

### achievement.js
- `achStats` — 成就統計（stars、title、loginDays、unlockedAchievements）
- 稱號路徑：換算LV1 → LV10（門檻：0/10/30/60/100/150/200/300/400/500★）
- `ACH_DEFS` — 4 個標準成就：答對題數、完成輪數、最高連擊、登入天數
- `CAT_BADGES` — 3 個題域徽章：長度達人、時間高手、換錢達人（各題域累積 10★）
- Firestore 路徑：`students/{id}/stats/convertProfile`
- `checkAchievements()` — 解鎖新成就並 showToast
- `handleDailyLogin()` — 每日首次登入更新記錄
- `renderAchievementPage()` — 渲染成就頁 HTML

### init.js
- `window.load` 啟動序列：initFirebase → applyTheme → showPage('home')
- 從 `sessionStorage('hub_student')` 自動登入，載入 progress/convert
- `toggleLogoutMenu / sendLogout / goToSettings / goToAchievement` — Topbar 學生選單
- `renderSettingsAvatarGrid / selectSettingsAvatar` — 設定頁頭像

## 頁面清單

| 頁面 ID        | 說明                       |
|---------------|----------------------------|
| `home`        | 三大題域卡片選擇             |
| `select`      | 題型 + 難度選擇，開始按鈕    |
| `game`        | 題目 + 虛擬鍵盤 + 統計列    |
| `result`      | 星評 + 分數 + 再練/換題/回首頁 |
| `settings`    | 頭像 / 主題 / 音效           |
| `achievement` | 成就軌道 + 題域徽章          |
