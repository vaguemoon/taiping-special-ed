# 上學趣 — 說明手冊

> 適用版本：2026-04 目前主版本

---

## 整體架構概覽

```
taiping-special-ed/
│
├── index.html              ← 入口頁（學生登入 + 科目 Hub）
├── admin.html              ← 教師後台
├── super-admin.html        ← 系統管理員後台
├── super-admin-login.html  ← 管理員登入頁
│
├── shared.css              ← 全域樣式（所有頁面共用）
├── shared.js               ← 共用函式庫（Firebase、音效、主題、Toast）
├── auth.js                 ← 學生 / 教師登入、註冊、Google 登入、登出
├── hub.js                  ← Hub 渲染、科目卡片、個人設定、班級加入、系統公告
│
├── admin/
│   ├── init.js             ← 教師後台啟動、Auth 驗證、分頁切換
│   ├── classes.js          ← 班級管理（建立 / 刪除 / 邀請碼）
│   ├── overview.js         ← 班級總覽（學生列表、統計）
│   ├── students.js         ← 學生詳細資料、進度查閱、字元狀態修改
│   ├── curriculum-ui.js    ← 課程版本手風琴列表、折疊面板
│   └── curriculum-data.js  ← 課程 Firebase CRUD、CSV 匯入、手動新增
│
├── super-admin/
│   ├── init.js             ← 管理員後台啟動、Auth 驗證
│   ├── overview.js         ← 全站統計總覽
│   ├── accounts.js         ← 教師帳號管理（封鎖 / 解鎖）
│   ├── invites.js          ← 邀請碼管理
│   └── settings.js         ← 維護模式、系統公告設定
│
└── chinese/
    ├── index.html          ← 國字練習 App（在 iframe 內執行）
    ├── chinese.css         ← 國字 App 專用樣式
    └── js/
        ├── nav.js          ← App 內頁面切換與返回歷史
        ├── state.js        ← 全域狀態、Firestore 讀寫、HanziWriter 工具函式
        ├── voice.js        ← 文字轉語音（Web Speech API）
        ├── curriculum.js   ← 課程版本 / 冊別 / 課次選擇、自由輸入生字
        ├── menu.js         ← 字卡主畫面、進度條、考試選字模式
        ├── char-info.js    ← 萌典 API 查詢（注音、部首、筆畫、造詞字義）
        ├── practice.js     ← 筆順練習、單字考試、視窗 resize 重建
        ├── dictation.js    ← 手寫默寫（canvas 手繪、自評）
        ├── exam.js         ← 多輪正式考試引擎（出題、提示、輪次結算）
        ├── achievement.js  ← 成就系統（星星、稱號、成就解鎖、頁面渲染）
        └── init.js         ← App 啟動初始化（Firebase、進度載入、教師推送）
```

**頁面關係：**
```
index.html  （學生登入 Hub）
  └── <iframe> → chinese/index.html  （國字練習）

admin.html         （教師後台，獨立頁面）
super-admin.html   （管理員後台，獨立頁面）
```
