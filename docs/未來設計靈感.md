# 「湊數趣」：
    此部分的練習重點不在於算多難的題目，而是在於**「看到數字組成，不經過大腦計算就跳出答案」**。
    ```
    ##   基礎補數（湊十與湊百）：湊十反射： 這是所有進位加法的靈魂。練習給予 $1-9$ 的任一數，學生需在 0.5 秒內反應出補數（例如：看到 $6$ 反射 $4$）。
    ```
    ##    湊百練習： 針對整十數的湊百（如：$30$ 對 $70$），以及尾數為 5 的湊百（如：$25$ 對 $75$）。跨十/跨百的拆解反射：
    ```
    ##    拆數練習： 例如 $8 + 5$，機械式練習會訓練孩子看到 $5$ 立刻拆成 $(2 + 3)$，以便與前面的 $8$ 湊成 $10$。減法退位反射： 看到 $13 - 8$，直覺反射出 $3 + (10 - 8) = 5$，而不是從 $13$ 往回數。
    ```
    ##    整十與整百的快速加減：例如 $47 + 30$ 或 $125 - 20$。練習讓學生在大腦中只撥動「十位數」或「百位數」珠子的能力，這對估算與日常購物非常實用。
# 「換算趣」：建立單位間的「位移」感
    換算的難點在於「進位制的不統一」（有十進位、六十進位等）。練習應著重於單位名稱與數字縮放的掛鉤。
    
    長度單位的「直覺縮放」 (十進位系)：毫米 (mm) 與 公分 (cm)： 建立「去一個 0」或「多一個 0」的機械反應。公尺 (m) 與 公分 (cm)： 建立「兩位數」位移的空間感（例如：$1\text{m } 5\text{cm}$ 絕對不是 $15\text{cm}$）。

    時間單位的「時鐘化」練習 (非十進位系)：60 進位常數： 熟記 $1/4$ 小時是 $15$ 分、$1/2$ 小時是 $30$ 分、$3/4$ 小時是 $45$ 分。跨單位轉算： 練習將 $90$ 分鐘快速反射為 $1$ 小時 $30$ 分，或是 $120$ 秒反射為 $2$ 分鐘。這需要對 $60$ 的倍數有機械式的熟悉度。

    貨幣與面額的組合：換錢反射： 練習「1 張 100 元可以換幾個 10 元？」或「50 元硬幣可以換幾個 5 元？」。湊錢練習： 給定一個金額（如 $73$ 元），學生要能最快速度反應出最精簡的硬幣組合（1 個 $50$、2 個 $10$、3 個 $1$）。
