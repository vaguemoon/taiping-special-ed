## 整體架構概覽

```
QuEDU/
│
├── index.html              ← 入口頁（角色選擇 + 教師登入）
├── login.html              ← 學生登入 / 註冊
├── hub.html                ← 學生學習中心（登入後才能進入）
├── admin/index.html        ← 教師後台（獨立頁面）
├── super-admin/index.html  ← 系統管理員後台
├── super-admin/login.html  ← 管理員登入頁
│
├── shared/

│
├── src/
│   ├── index.css           ← 入口 / 登入 / Hub 樣式（三頁共用）
│   ├── auth.js             ← 學生 / 教師登入、註冊、Google 登入、登出、系統設定檢查

│
├── admin/
│   ├── init.js             ← 教師後台啟動、Auth 驗證、分頁切換

│   ├── overview.js         ← 班級總覽（學生列表、統計）
│   ├── students.js         ← 學生詳細資料、進度查閱、字元狀態修改
│   ├── curriculum-ui.js    ← 課程版本手風琴列表、折疊面板
│   └── curriculum-data.js  ← 課程 Firebase CRUD、CSV 匯入、手動新增
│
├── super-admin/
│   ├── init.js             ← 管理員後台啟動、Auth 驗證
│   ├── overview.js         ← 全站統計總覽
│   ├── accounts.js         ← 教師帳號管理（封鎖 / 解鎖）
│   ├── invites.js          ← 邀請碼管理
│   └── settings.js         ← 維護模式、系統公告設定
│
└── apps/
    ├── learn/lang/chinese/     ← 練字趣（遊戲類）
    ├── learn/lang/recognize/   ← 認字趣（遊戲類）
    ├── learn/math/multiply/    ← 乘法趣（遊戲類）
    ├── quiz/word-to-reader/    ← 考卷報讀（工具類）
    └── quiz/chinese-quiz/      ← 語文練習（測驗類）
```

**頁面關係與導覽流程：**
```
index.html  （角色選擇）
  ├── [學生] → login.html （學生登入 / 註冊）
  │               └── 登入成功 → hub.html （學習中心）
  │                               ├── <iframe> → apps/learn/lang/chinese/
  │                               ├── <iframe> → apps/learn/math/multiply/
  │                               ├── <iframe> → apps/learn/lang/recognize/

admin/index.html       （教師後台，獨立頁面）
```

## 規劃中子 App

以下為未來預計加入的子 App，供設計參考：

### 湊數趣（數學 — 補數反射）

練習重點：**看到數字組成，不經計算即跳出答案**。

- **湊十反射**：給 1–9 任一數，0.5 秒內反應出補數（例：看到 6 反射 4）
- **湊百練習**：整十數的湊百（30 對 70）、尾數為 5 的湊百（25 對 75）
- **拆數練習**：例如 8 + 5，訓練看到 5 立刻拆成 (2 + 3) 以便湊十
- **減法退位反射**：看到 13 − 8，直覺反射 3 + (10 − 8) = 5
- **整十整百快速加減**：如 47 + 30、125 − 20，只撥動十位 / 百位珠子

### 換算趣（數學 — 單位換算）

練習重點：**單位間的「位移」感，不同進制各有記憶錨點**。

- **長度（十進位）**：mm ↔ cm（去一個 0）、cm ↔ m（兩位數位移）
- **時間（六十進位）**：熟記 ¼ 小時 = 15 分、½ = 30 分、¾ = 45 分；90 分鐘 ↔ 1 小時 30 分
- **貨幣與面額**：換錢反射（100 元 → 幾個 10 元）、最精簡硬幣組合（73 元 → 50 + 10×2 + 1×3）
