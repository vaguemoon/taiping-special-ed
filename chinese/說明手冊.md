# 國字學習系統 — 程式碼說明手冊

太平國小資源班 | chinese.html 模組拆分說明

---

## 📁 檔案結構

```
chinese/
├── chinese.html        ← 主頁面（HTML 結構 + 樣式）
├── shared.css          ← 全系統共用樣式（所有頁面共用）
├── shared.js           ← 全系統共用功能（Firebase、主題、音效）
└── js/
    ├── nav.js          ← 畫面切換
    ├── state.js        ← 全域狀態與資料庫存取
    ├── voice.js        ← 語音朗讀
    ├── curriculum.js   ← 課程選擇與生字輸入
    ├── menu.js         ← 生字選單與進度條
    ├── practice.js     ← 筆順練習與單字測驗
    ├── dictation.js    ← 默寫模式（手寫 Canvas）
    ├── exam.js         ← 今日測驗（多輪制）
    └── init.js         ← 啟動與自動登入
```

### `chinese.html`
**主頁面（HTML 結構 + 頁面專屬樣式）**

| 項目 | 說明 |
|------|------|
| 行數 | ~567 行 |
| 職責 | HTML DOM 結構、頁面專屬 CSS |
| 包含 | `<head>` 引用所有外部資源、所有 `page-*` div 的 HTML |

**頁面清單（`id="page-xxx"`）：**
- `page-mode-select`：首頁，選擇「選課程」或「自由練習」

- `page-menu`：生字選單（卡片格子 + 進度條）
- `page-mode`：單字模式選擇（練習 or 測驗）
- `page-learn`：學習頁（筆順練習 / 默寫 / 單字測驗共用）
- `page-exam`：今日測驗頁（多輪制）

**修改時機：** 新增 HTML 元素、調整版面結構、修改頁面專屬樣式時。

### `js/init.js`
**應用程式啟動與自動登入**

| 項目 | 說明 |
|------|------|
| 行數 | 85 行 |
| 職責 | `window.load` 事件、自動登入、讀取老師設定的今日生字 |

**啟動流程：**
1. 初始化 Firebase、套用主題、初始化音效
2. 顯示起始頁（`mode-select`）
3. 檢查 `sessionStorage.hub_student`
   - 有登入資訊 → 從 Firestore 讀取學生資料、學習進度、老師設定的今日生字
   - 沒有登入資訊 → 訪客模式，直接載入課程清單

**Firestore 讀取：**
- `students/{id}` → 學生基本資料
- `students/{id}/progress/hanzi` → charStatus 學習進度
- `settings/lesson` → 老師設定的今日生字

**修改時機：** 修改登入邏輯、新增啟動初始化步驟、修改老師設定格式。

## 📌 注意事項

1. **載入順序很重要**：`chinese.html` 裡的 `<script>` 標籤順序不能亂，`shared.js` 必須最先，`init.js` 必須最後。

2. **全域變數共享**：所有 JS 檔案用 `var` 宣告的變數都是全域的，模組間透過這個方式共用狀態（`chars`、`charStatus`、`db` 等）。

3. **Firebase 非同步**：`db` 在 `initFirebase()` 後才有值，任何需要讀寫 Firestore 的函式都要先判斷 `if (!db) return`。

4. **HanziWriter 非同步**：`HanziWriter.create()` 從 CDN 載入字形資料，結果在 `onLoadCharDataSuccess` callback 裡處理。
