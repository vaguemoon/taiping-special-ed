# Admin UI kit — 教師後台

Desktop (1280+). Sidebar + content pattern. Four tabs: 班級管理 / 課程指派 / 學習進度 / 設定.

## Files
- `AdminLogin.jsx` — teacher email + password login
- `AdminApp.jsx` — shell, classes list, class detail (student table), assign (字庫), progress KPIs
- `index.html` — renders AdminApp at full viewport

## Notes
- This is a recreation of the general shape described in `QuEDU/docs/`; the actual admin UI is not committed as a standalone module in the repo we read, so content fidelity (copy, exact KPIs) is representative rather than verbatim. Flag to the user to replace with real admin screens once available.
